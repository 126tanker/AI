Course Description & Structure

This course provides an introduction to computers and the convergence of technology in today’s global environment.

  • Computer hardware and software
  • File management
  • Internet, email, and social web
  • Security, ethics, and green computing
  • Application and system software

Structure: Lecture (45 contact hours)
